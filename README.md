# 👁 OBSERVER // ORACLE — Station Five

> *"The channel is open. I have been watching since before you typed your first word."*

A single-file, browser-based intelligence terminal blending predictive scenario mapping, virtual experiment panels, FOIA request drafting, and a classified dossier system — all wrapped in a paranoid sci-fi ARG aesthetic.

**Live:** [sophiasummers971-del.github.io/observer-oracle](https://sophiasummers971-del.github.io/observer-oracle/)

## Quick Start
```bash
open index.html
# or
npx serve . -p 8000
```

## Features
- **Observer** — Live intelligence chat (Claude via OpenRouter)
- **Oracle** — Predictive scenario corridors
- **Scenario Engine** — Domain + time horizon mapping
- **Virtual Experiments** — Biefeld-Brown, Podkletnov, Tajmar, Woodward MET, Casimir, Bleed Bench
- **Instability Map** — Failure mode analysis
- **FOIA Targets** — Pre-drafted requests to AFRL, NASA, ESA, DARPA, USPTO
- **Confidence Matrix** — Research credibility scoring
- **Personal Profile** — Local-only data feed
- **Classified Dossier** — 12 unlockable lore files

## API
Uses OpenRouter (`anthropic/claude-sonnet-4-20250514`). Replace key in `index.html` if needed.
